## 1. 产品结构（MVP）

MVP 先把两件事做扎实：

- **有知（Wiki）**：可检索、可分享、可被搜索引擎抓取的知识库
- **相辅（Forum）**：像小红书一样的帖子流，核心交互只做 **发帖 + 评论**

潮汕话特色命名（UI 用语）：

- 相辅 = 帖子
- 有知 = 百科
- AI来 = AI（P1 可选）
- 胶己 = 我的
- 劳热 = 时间线（P1 可选）
- 辣辣嗦 = 地图（P1 可选）

明确不做（P0）：私信/聊天、匹配/接单、志愿者体系、审核工作台、信息卡片独立功能。
